= Test::Unit::Full

* http://rubyforge.org/projects/test-unit/

== DESCRIPTION:

Test::Unit::Full - A meta package to use all Test::Unit extensions.

== FEATURES/PROBLEMS:

* This provides all Test::Unit extensions just one require line.

== INSTALL:

* sudo gem install test-unit-full

== USAGE:

  require 'test/unit/full'
